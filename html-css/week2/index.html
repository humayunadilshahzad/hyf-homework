WEB DEVELIPMENT IS AN INDUSTRY

Examples of dramatic transformation in comunication and commerce led by web development include e-commerce. Online auction sites such as eBay has changed the way consumers find and purchase goods and services. Online retailers such as Amazon.com and Buy.com(among many others) have transformed the shoping and bargain-hunting experience of many customers.

PRACTICAL WEB DEVELOPMENT

Since the commercialization of the web, web development has been growing industry. The growth of this industry is being driven by businesses wishing to use their website to sell the products and services to customers.[1]

There is open source software for web development like BerkeleyDB, GlassFish, LAMP(Linux, Apache, MySQL,PHP) stack and Perl/Plack. This kept the cost of learning web development to a minimum. Another contributing factor to thegrowth of the industry has been the rise of easy-to-use WYSIWYG web-development software such as Adobe Dreamweaver, BlueGriffon and Microsoft Visual Studio. Knowledge of Hyper Text Markup Language(HTML)or of programming languages is still required to use the such software, but the basics can be learned and implemented quickely with the help of help files, technical books, internet tutorials,or face-to-face training.

An ever-growing set of tools and technologies have helped developers build more dynamic and interactive websites. Further, web developers now help to deliver applications as web services which were traditionally only availabel as applications on a desk-based computer.This has allowed for many oppertunities to decentralize informationand media distribution. Examples can be seen with the rise of cloud services such as Adobe Creative Cloud, Dropboxand Google Dics. These web services allow users to interact with applications from many locations, instead of being tied to a specific workstaion for their applicatoin environment.
